# Weblogアプリの作成
## はじめに
Web上で動作するブログアプリ（Weblogアプリ）を作成しました。デプロイを行うような正規品ではなく、学習用に作成したものなので機能は「ログイン・記事の新規投稿/更新/削除」といったシンプルなものに限定して作成しました。
今回の学習についての詳細は以下のページにて紹介しています。  
https://troubled-event-601.notion.site/Weblog-App-2bd465c59ef380fb8549c6f145363298?source=copy_link

## 目的
今回このアプリを作成するにあたっては大きく次の3つの内容を学びました。  
1. セッション管理の理解  
Cookieとセッションについて理解を深めることを１つ目標に取り組みました。また、ログイン機能の実装を通じてセッション管理が実際のWEBアプリでどのように使われているかも合わせて学習しました。
2. CRUD機能の実装  
WEBアプリの基本4機能である「**C**reate(作成) / **R**ead(参照) / **U**pdate(更新) / **D**elete(削除)」を実装し、アプリ開発に必要な最低限のスキルを習得しました。
3. MVCモデルでの実装  
フレームワークの内部構造のイメージ取得を目的に、MVCモデルをベースにフルスクラッチ開発を行いました。Laravel学習前に一度フルスクラッチで実装することで、フレームワークを魔法の道具としてではなく原理を理解して使用できる状態を目指して取り組みました。

## 使用技術
フロント側：html、css（scss）  
バッグエンド側：PHP 8.2、MySQL 8.0、apache  
環境構築：Docker 28.3.3  
その他：セッション管理、パスワードのハッシュ化によるセキュリティ対策
